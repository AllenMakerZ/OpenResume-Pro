# 导出与使用注意事项

## 1. 打印 / 另存为 PDF 怎么使用？
  - 使用浏览器菜单中的 **“打印 / Print…”**，或快捷键 `Cmd+P` / `Ctrl+P`。
  - 在打印对话框中选择 **“另存为 PDF / Save as PDF”** 作为目标。
      - **边距**设置为 **“最小值”**：可以避免文字贴边。
      - 纸张选择 **A4**，缩放请选择 **100% / 实际大小**。
      - 关闭浏览器自带的页眉、页脚。

## 2. 下载为 PNG 的渲染逻辑
  - 以右侧 A4 预览区域的**固定宽度**为基准进行渲染，并使用 **2 倍分辨率** 导出图片（假设简历内容小于 A4 尺寸，则分辨率为**1588px x 2246px**）。
  - 高度随内容自适应，可超过一页 A4，这更接近“一张截图”，适合用于在线投递或预览。

## 3. 下载为 PDF（图片版）的渲染逻辑
  - 生成的 PDF 本质上是“整页截图版 PDF”，导出后不会再重新进行文字排版或分页。

## 4. 内容超出一页时如何调整？
  - 建议通过 **减少内容量**，并在编辑器中适当调整 **字体大小**、**行间距**、**页边距**（顶部按钮），来控制整体高度。

## 5. 预览与打印不一致时怎么办？
   - 不同浏览器和系统的打印引擎略有差异，可能出现页面预览与打印预览不完全一致的情况。
   - 遇到这种情况时，请以实际导出的 **PDF 文件** 或系统打印对话框中的 **打印预览效果** 为准。

## 打印建议：
  - 打印时请选择 **“实际大小 / 100%”**，关闭浏览器的额外页眉页脚。
  - 纸张尺寸请选择 **A4**，方向为 **纵向（Portrait）**。
- **推荐浏览器**：建议使用 **桌面版本 Chrome / Edge / Safari**，确保对 `clip-path`、字体与打印样式有较好的支持。

## 项目简介

这是一个 **模块化简历生成器**，左侧为数据编辑面板，右侧为实时 A4 预览区域，支持对各模块进行增删改、拖拽排序（如果已开启）、自定义标题与显示 / 隐藏控制，适合快速生成视觉统一、结构清晰的中文简历。

- **基础信息**：填写姓名、联系方式、个人链接等内容，输入内容后会实时同步到右侧 A4 预览区。
- **教育经历**：添加或编辑你的学历条目，可以自定义模块标题、排序，也支持随时隐藏显示。
- **工作经历**：录入实习、工作经历，支持条目的新增、删除与标题自定义，如无需求可选择隐藏。
- **项目经历**：添加参与过的项目经验，支持自定义标题与显示顺序，不需要时可直接隐藏。
- **其他信息**：用于技能、证书、语言等补充说明，有内容即可显示，可自定义模块标题与显示状态。
- **个人总结**：添加简要的自我介绍或总结，能自定义该模块的名称及是否展示。
- **模块管理**：集中操作所有模块，可一键删除、排序、显示/隐藏，也支持添加自定义模块。固定模块被误删时，可在这里恢复显示。

- **预览功能**：页面右侧实时展示 A4 格式排版效果，所有编辑和设置都会即时反映在预览区，便于随时检查最终排版。
- **重置功能**：如需从头开始编辑，可在顶部点击重置按钮，将所有内容恢复为初始示例数据，请注意此操作不可撤销。

- **导出简历**：点击页面顶部的“导出为 PDF”或“导出为 PNG”按钮，导出的内容与右侧当前预览一致，适用于投递和打印。

## 技术架构与实现说明

- **框架与语言**：
  - 使用 **React 19** + **TypeScript** 构建单页应用（SPA）。
- **样式与布局**：
  - 基于 **Tailwind CSS** 的原子化样式进行布局和排版。
  - A4 预览容器在 `index.html` 中定义基础尺寸与打印样式，配合组件内的样式实现高保真预览。
- **组件与状态管理**：
  - `App.tsx`：顶层组件，维护集中化的 `ResumeData` 状态，并负责导出逻辑。
  - `Editor`：左侧编辑器组件，负责修改和管理简历数据。
  - `ResumePreview`：右侧 A4 预览组件，只负责渲染当前 `ResumeData`。
  - `SectionHeader`：独立的标题展示组件，内部使用 `clip-path` 实现黑色梯形块与灰色装饰块的几何效果。
- **导出实现**：
  - 使用 `dom-to-image` 从预览区域直接生成位图，再结合 `jsPDF` 生成标准 A4 PDF，最大程度保证导出与屏幕预览一致。
